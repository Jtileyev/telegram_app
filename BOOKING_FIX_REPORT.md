## 🔧 Технические детали

### Race Condition Protection

**До**:
```python
# Проверка
available = check_apartment_availability(...)
# ВРЕМЯ МЕЖДУ ПРОВЕРКОЙ И СОЗДАНИЕМ = УЯЗВИМОСТЬ
if available:
    create_booking(...)  # Может создать конфликт
```

**После**:
```python
conn.execute("BEGIN IMMEDIATE")  # Начало транзакции
# Проверка и создание атомарны
overlapping = conn.execute("""SELECT COUNT(*)...""")
if overlapping['count'] > 0:
    conn.rollback()
    raise ValueError(...)
create_booking_in_transaction(...)
conn.commit()  # Конец транзакции
```

### Валидация дат

```python
# Парсинг и валидация формата
check_in_date = dt.strptime(check_in, '%Y-%m-%d').date()
check_out_date = dt.strptime(check_out, '%Y-%m-%d').date()

# Проверка что не в прошлом
if check_in_date < today:
    raise ValueError("Check-in date cannot be in the past")

# Проверка порядка
if check_out_date <= check_in_date:
    raise ValueError("Check-out date must be after check-in date")

# Минимальная длительность
if (check_out_date - check_in_date).days < 1:
    raise ValueError("Minimum booking duration is 1 day")
```
