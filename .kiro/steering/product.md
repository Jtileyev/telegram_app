# Product Overview

**Astavaisya** (Аставайся) is a Telegram bot platform for daily apartment rentals in Kazakhstan.

## Core Features

- **Telegram Bot** (Python/aiogram): User-facing interface for searching, booking, and managing apartment rentals
- **Admin Panel** (PHP): Web dashboard for managing apartments, bookings, users, and platform settings

## User Capabilities
- Bilingual support (Russian/Kazakh)
- Search apartments by city and district
- Interactive calendar with availability
- Booking with landlord notifications
- Favorites and booking history
- Reviews and ratings
- Promotion system (N-th booking free)

## Landlord Features
- View partnership conditions
- Submit connection requests
- Receive booking notifications

## Business Model
- Platform fee: 5% commission on bookings
- Direct payment between renters and landlords
