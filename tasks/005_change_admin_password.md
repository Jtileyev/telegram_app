# Задача 005: Сменить пароль администратора

## Приоритет: 🔴 Высокий

## Описание
В системе используются стандартные учётные данные `admin/admin123`. Нужно сменить пароль на безопасный.

## Шаги выполнения

### 1. Сгенерировать новый пароль
Используйте надёжный пароль минимум 12 символов:
- Заглавные и строчные буквы
- Цифры
- Специальные символы

### 2. Захешировать пароль
В терминале выполнить:
```bash
source venv/bin/activate
python3 -c "import bcrypt; print(bcrypt.hashpw(b'YOUR_NEW_PASSWORD', bcrypt.gensalt()).decode())"
```

Или через PHP:
```bash
php -r "echo password_hash('YOUR_NEW_PASSWORD', PASSWORD_BCRYPT);"
```

### 3. Обновить пароль в БД
```bash
sqlite3 database/rental.db
```

```sql
UPDATE users 
SET password = '$2b$12$...(ваш_хеш)...' 
WHERE email = 'admin@astavaisya.kz' OR username = 'admin';
```

### 4. Проверить вход
1. Открыть админку: http://localhost:8080
2. Войти с новым паролем

## Альтернативный способ
Через админку (если уже есть доступ):
1. Войти в админку
2. Перейти в "Пользователи"
3. Редактировать админа
4. Изменить пароль через форму

## Критерии завершения
- [ ] Пароль изменён на безопасный
- [ ] Вход в админку работает с новым паролем
- [ ] Старый пароль `admin123` не работает

## Важно!
- Сохраните новый пароль в надёжном месте (менеджер паролей)
- НЕ храните пароль в коде или git
