# Задача 003: Модерация отзывов

## Приоритет: 🟡 Средний

## Описание
Отзывы публикуются сразу без проверки. Нужно добавить статус модерации и возможность одобрять/отклонять отзывы в админке.

## Шаги выполнения

### 1. Изменить схему БД (если нужно)
В таблице `reviews` уже есть поле `is_visible` (BOOLEAN DEFAULT 1).
Можно использовать его для модерации:
- `is_visible = 0` - на модерации / скрыт
- `is_visible = 1` - опубликован

Или добавить новое поле для более точного статуса:
```sql
ALTER TABLE reviews ADD COLUMN moderation_status TEXT DEFAULT 'pending';
-- Значения: 'pending', 'approved', 'rejected'
```

### 2. Изменить создание отзыва в боте
В `bot/database.py` функция создания отзыва:
- По умолчанию ставить `is_visible = 0` (на модерации)
- Или `moderation_status = 'pending'`

### 3. Обновить админку reviews.php
- Показывать статус модерации
- Добавить фильтр по статусу (все / на модерации / одобренные)
- Добавить кнопки "Одобрить" / "Отклонить"
- При одобрении: `is_visible = 1`
- При отклонении: `is_visible = 0` или удаление

### 4. Изменить отображение отзывов в боте
В `bot/database.py` функция `get_apartment_reviews`:
- Показывать только `is_visible = 1`

## Файлы для изменения
- `database/schema.sql` - возможно добавить поле статуса
- `bot/database.py` - создание и получение отзывов
- `admin/reviews.php` - интерфейс модерации

## Критерии завершения
- [ ] Новые отзывы требуют модерации
- [ ] В админке видны отзывы на модерации
- [ ] Можно одобрить/отклонить отзыв
- [ ] В боте показываются только одобренные отзывы
- [ ] Пользователь видит статус своего отзыва

## Опционально
- Уведомлять пользователя об одобрении/отклонении отзыва
- Показывать причину отклонения
