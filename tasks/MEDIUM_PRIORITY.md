## 🟡 5. Оптимизация запросов (N+1 проблема)
**Приоритет**: Средний (производительность)
**Статус**: ❌ Не исправлено

### Проблема
```python
# Множественные запросы в цикле
for booking in bookings:
    apartment = get_apartment(booking.apartment_id)  # N+1!
    user = get_user(booking.user_id)  # N+1!
```

### Решение
1. Использовать JOIN запросы
2. Добавить в database.py:
   - `get_bookings_with_details()`
   - Eager loading связанных данных
3. Кеширование частых запросов

### Оценка времени
6 часов
